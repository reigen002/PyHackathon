# Virtual Assistant Project

## Overview
This project is an AI-powered Virtual Assistant designed to help users with basic automation tasks. The assistant can open specific applications on command and provide simple interactions through voice and text input.

## Features
- **Application Launcher**: Opens predefined applications on the user's system.
- **Basic Command Execution**: Executes basic system commands.
- **Voice Command Support**: Users can interact with the assistant using voice input.

## Technologies Used
- **Programming Language**: Python
- **Frameworks/Libraries**: SpeechRecognition, Pyttsx3 (for text-to-speech), Flask (for future expansion)
- **APIs Used**: None (for now)




## Usage
- Start the virtual assistant.
- Use voice or text commands to open applications like Notepad, Calculator, or a web browser.
- Modify `config.py` to add more applications.

## Future Enhancements
- **Improved Speech Recognition**: Enhance accuracy for better voice command interpretation.
- **Task Automation**: Add features for scheduling tasks and reminders.
- **API Integrations**: Expand functionality with third-party services.
